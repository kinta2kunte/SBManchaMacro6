SBManchaMacro概要説明

1．はじめに

　当プログラムはShadowbane（以下SB）ゲーム上のキーボード操作を擬似的に行い、自動動作が行える様にします。
　つまりSBゲーム上に於いてキーボードで動作させられる全ての操作の自動化を行えます。　
　通常のmob 攻撃用の繰り返し動作以外に、自己buff、及び各種potタイミングを指定する事が可能です。

2．使い方

　基本サンプルとして、Warlock、ドルイド、トラックを初期状態にて選択できます。
　Warlock で使用したい場合は初期で選択されていますので、右の編集ボタンを押下して下さい。

　■編集（設定画面）
　設定画面では以下の設定が可能です。

　キー設定 ... 「はじめに」で記したように当プログラムはSB専用のキーボード疑似操作を行いますので、割り当てキーの設定を行います。
　　　　　　　　例えばHotKeyの"F1"キーにSBスキル・自己INT Buffを割り当てている場合、以下の様に設定します。

　　　　　　　　スキル名称： 自己Int Buff  　※名称は分かりやすい様に自由編集下さい
　　　　　　　　キーコード： F1　　　　　　　※該当箇所にフォーカスが当たっている状態で任意のキーを押下下さい。
　　　　　　　　キャスト時間： 4.6           ※ここでは自己INT Buffのキャスト時間を秒単位にて指定

　　　　　　　　他 Warlock の基本的なスキル、及びターゲット等の設定はご自分のSB設定に合わせるか編集ください。

　攻撃サイクル設定 ...  上記キー設定のスキルを具体的に指定してください。
　　　　　　　　　　　　当該設定したスキルを順番に無限ループさせます。

　自己buff設定 ... SBの Warlockや各職業には各種自己buffがあり、ある一定時間（効果時間）事にかけ直す必要があります。
　　　　　　　　　 「割り込み周期時間」に効果時間を設定する事により、当該時間経過時に一連の設定スキルを発行させます。
　　　　　　　　　 必要ない場合は未設定でも構いません。

　pot buff設定 ... SBで長時間自動操作を行いたい場合は、各種potの利用が必須となります。
　　　　　　　　　 pot(conc)効果時間は3600秒でかけ直す必要があります。
　　　　　　　　　 「割り込み周期時間」に効果時間を設定する事により、当該時間経過時に一連の設定スキルを発行させます。
　　　　　　　　　 ※予めホットキーに掛けたいpotを設定しておいて下さい。
　　　　　　　　　 必要ない場合は未設定でも構いません。

　最低限の設定はサンプル＞トラックとなります。
　サンプルの「トラック」はSBに於けるトラックスキル（一定範囲内のプレイヤー表示スキル）のマクロになります。
　キー設定はF6:トラック設定と10秒ウエイト（キー無しでキャスト10秒設定 = 10秒なにもしない）のみです。
　SBゲーム上にてホットキーF6にトラックスキルのアイコンを設定し、F6キーでトラックスキルが発動できるように設定してください。
　攻撃サイクル設定で トラック＞10秒ウェイトを実行中永遠に繰り返します。

　又、トラックは10秒毎に定時間隔で動作させたいので、トラック動作を定時サイクル・自己buffに設定しても問題ありません。
　その場合、攻撃サイクルは削除し、自己buffの定時間隔を10秒にセットし、Trackを追加ボタンで設定してください。

　このように
　＞通常無限ループ動作＝攻撃サイクル
　＞一定時間経過にて動作＝定時サイクル（自己buff/自己buff2/その他/pot buff設定）
　を設定すると、攻撃サイクルに設定したキーを無限ループで繰り返し、一定時間経過時に定時サイクルに設定された一連の動作キーを
　割り込みで動作させます。

　■オプション設定

　オプションは現在の所以下の設定が可能です。

　コマンド間wait設定 ... マナを使用するスキルを高速で回すと、マナリカバリーが不足してしまう場合があると思います。
　　　　　　　　　　　　 これを回避する為、スキル発行間に何もしない時間をここで設定します。
　　　　　　　　　　　　 当該値を長くするとマナ回復量が増えますので調整用に使用して下さい。
　常に最前面に表示 ... 当オプションをチェックするとフォーカスが他ウィンドウに移動しても最前面に表示されるようになります。
　　　　　　　　　　　 利用環境に応じて設定下さい。

　ログ出力 ... 解析用ですので通常offで利用下さい。
